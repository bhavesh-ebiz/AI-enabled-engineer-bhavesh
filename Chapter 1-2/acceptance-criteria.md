# Acceptance criteria:

1. If tab is inactive for defined time period or default time, it should be automatically assigned to the specificed inactive group.
2. Inactive group tab should be visible to the user properly.
3. User should allow to remove the tab from the group.
4. Visually consistent with Chrome’s tab grouping system.
5. After interacting the inactive tab, it should be moved outside of the group.
6. After removing the tab from the group, it should be automatically moved to the default group.
